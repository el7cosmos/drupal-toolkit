# Drupal Toolkit Changelog

## [Unreleased]

## [0.3.0] - 2025-12-14

### Added

- Provide `attributes` variable for twig files.
- Provide `componentMetadata` variable for component's Twig file.

## [0.2.0] - 2025-12-13

### Added

- Declare `settings.php` predefined variables
- Twig namespace support for twig files under `<module|theme>/templates` directory

## [0.1.0] - 2025-12-06

### Added

- SDC support in render arrays

## [0.0.1] - 2025-12-02

### Added

- Initial scaffold created
  from [IntelliJ Platform Plugin Template](https://github.com/JetBrains/intellij-platform-plugin-template)
- Single directory component (SDC) support
